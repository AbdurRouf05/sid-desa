Abstrak & Latar Belakang

Kerja Praktik ini dilaksanakan di Kantor Desa Sumberanyar dengan fokus pada pengembangan sistem administrasi desa. Sebelumnya, proses manual rentan terhadap kesalahan, keterlambatan pelayanan, dan fragmentasi data kependudukan.

Tujuan utama adalah digitalisasi untuk meningkatkan efisiensi operasional, akurasi data, dan transparansi keuangan melalui Sistem Informasi Desa (SID) terintegrasi.

Target Pemecahan Masalah

  • Menggantikan administrasi manual yang lambat.
  • Integrasi data warga, persuratan, dan keuangan.
  • Antarmuka responsif berbasis "Desa Teal".
  • Meminimalisir human error.

Metodologi & Tech Stack

Metode Agile (Scrum)

Pelaksanaan iteratif dengan sistem kerja remote dan hybrid. Evaluasi rutin setiap minggu untuk mengakomodasi perubahan kebutuhan Desa.

Next.js & Tailwind

Pemanfaatan Next.js untuk Frontend dan Backend dengan styling Tailwind CSS untuk merancang identitas visual yang responsif.

PocketBase

Digunakan sebagai backend mandiri (BaaS) penyedia sistem manajemen basis data dan API yang ringan serta real-time.

Tahapan Pelaksanaan

1. Observasi & Analisis Minggu 1-2

Wawancara perangkat desa, identifikasi masalah manual, perancangan ERD, UI/UX, inisiasi repository.

2. Pengembangan Fitur Utama Minggu 3-6

Portal publik, pengaduan, manajemen kependudukan, Surat Pintar (jsPDF), Buku Kas Umum, integrasi grafik (Recharts).

3. Penyempurnaan Sistem Minggu 7-8

Export Excel (XLSX), kalkulasi PPN/PPh, logic inventaris aset, dan penyusunan draft Manual Book.

4. UAT & Serah Terima Minggu 9

Quality Assurance, security test, User Acceptance Test (UAT) bersama perangkat desa, dan serah terima sistem.
